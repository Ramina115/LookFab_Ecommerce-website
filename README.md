# LookFab - Full Stack E-Commerce Website

 Overview

LookFab is a full-stack fashion e-commerce web application developed to provide a seamless online shopping experience. The platform allows users to browse products, manage their shopping cart, securely complete purchases through the Khalti Payment Gateway, and manage their accounts. The project demonstrates practical implementation of frontend, backend, and database technologies in a real-world e-commerce system.
Features

- User Registration & Login
- Secure Authentication
- Product Browsing
- Product Search & Filtering
- Shopping Cart
- Checkout System
- **Khalti Payment Gateway Integration**
- Order Management
- Admin Dashboard
- Product Management (CRUD)
- Responsive Design
